# 🎨 মেস ফাইন্ডার — সম্পূর্ণ ডিজাইন স্পেসিফিকেশন গাইড

> **ডিজাইনারের জন্য নোট:** এই ডকুমেন্টে প্রজেক্টের প্রতিটি ডিজাইন এলিমেন্ট — ফন্ট, কালার, এফেক্ট, কম্পোনেন্ট স্টাইল, স্পেসিং, অ্যানিমেশন — বিস্তারিত দেওয়া আছে। ডিজাইন করার সময় এই স্ট্যান্ডার্ডগুলো মেনে চললে আউটপুট সামঞ্জস্যপূর্ণ হবে।

## ১. ফন্ট সিস্টেম (Typography)

### ফন্ট পরিবার
| ভূমিকা | ফন্ট নাম | ব্যবহার |
|---|---|---|
| **বাংলা টেক্সট** | `Hind Siliguri` | সব বাংলা লেখা (weight: 300, 400, 500, 600, 700) |
| **ইংরেজি টেক্সট** | `Inter` | ইংরেজি লেখা, সংখ্যা, কোড |
| **মোনোস্পেস** | `JetBrains Mono` | রেফারেন্স আইডি, কোড স্নিপেট |

### ফন্ট সাইজ হায়ারার্কি
```
H1 (হিরো টাইটেল):     text-3xl md:text-5xl  →  30px / 48px, font-extrabold
H2 (সেকশন টাইটেল):     text-2xl md:text-3xl  →  24px / 30px, font-bold
H3 (কার্ড টাইটেল):      text-base / text-sm   →  16px / 14px, font-bold
বডি টেক্সট:            text-sm               →  14px, font-normal
ছোট টেক্সট:            text-xs               →  12px, font-normal
ক্যাপশন/লেবেল:         text-[11px]           →  11px, font-medium
```

### ফন্ট ওয়েট গাইডলাইন
- **Bold (700):** হিরো টাইটেল, কার্ড নাম, প্রাইস
- **Semi-bold (600):** সেকশন হেডিং, বাটন টেক্সট, লেবেল
- **Medium (500):** নেভিগেশন, ব্যাজ, ছোট লেবেল
- **Regular (400):** বডি টেক্সট, ডেসক্রিপশন

### কী লেআউট প্যাটার্ন

#### ম্যাপ সার্চ পেজ (ডেস্কটপ — স্প্লিট ভিউ)

```
│ ফিল্টার   │       ইন্টারঅ্যাকটিভ ম্যাপ      │
│ প্যানেল   │       (৭০% স্ক্রিন)            │
│ (২৮০px)  │                              │
│          │                              │
├──────────┴──────────────────────────────┤
│ ফুটার                                   │
└─────────────────────────────────────────┘
```

#### ম্যাপ সার্চ পেজ (মোবাইল)
```
┌─────────────────┐
│ সার্চ বার (ফ্লোটিং) │
├─────────────────┤
│                 │
│   ফুল-স্ক্রিন ম্যাপ  │
│                 │
│                 │
├─────────────────┤
│ [ফিল্টার] [ম্যাপ/লিস্ট] │ ← বটম বার
```

#### ড্যাশবোর্ড লেআউট
```
┌─────────────────────────────────────────┐
│ হেডার (ইউজার নাম + ব্যাজ + লগআউট)       │
├──────────┬──────────────────────────────┤
│          │                              │
│ সাইডবার   │       মূল কন্টেন্ট এরিয়া       │
│ নেভিগেশন  │       (KPI / টেবিল / চার্ট)   │
│ (২০০px)  │                              │
│          │                              │
└──────────┴──────────────────────────────┘
মোবাইলে: সাইডবার → হরাইজন্টাল স্ক্রল ট্যাব
```

## ১৯. সামগ্রিক ডিজাইন ভাষা (Design Language Summary)

### ভিজ্যুয়াল স্টাইল
- **মডার্ন ফ্ল্যাট ডিজাইন** — সাবটল ডেপথ, নো নিউমরফিজম
- **ক্লিন ও মিনিমাল** — প্রচুর হোয়াইট স্পেস
- **এমেরাল্ড অ্যাকসেন্ট** — ব্র্যান্ড কালার সর্বত্র
- **বাংলা-ফার্স্ট** — সব টেক্সট বাংলায়, সংখ্যাও বাংলা সংখ্যায়

### এমোশনাল টোন
- **বিশ্বাসযোগ্য:** ভেরিফায়েড ব্যাজ, সিকিউরিটি সংকেত
- **অ্যাপ্রোচেবল:** নরম কালার, গোলাকার কোণ
- **দক্ষ:** পরিষ্কার লেআউট, দ্রুত অ্যাকশন
- **লোকাল:** বাংলা ভাষা, ঢাকার এলাকা নাম

### ডিজাইন DO's ও DON'Ts

✅ **করুন:**
- এমেরাল্ড গ্রিনকে প্রাইমারি অ্যাকসেন্ট হিসেবে ব্যবহার করুন
- সব টেক্সট বাংলায় রাখুন (সংখ্যাসহ)
- কার্ডে সাবটল শ্যাডো ব্যবহার করুন (হোভারে)
- সিট চার্টে রঙ-কোডেড স্ট্যাটাস ব্যবহার করুন
- প্রচুর হোয়াইট স্পেস রাখুন

❌ **করবেন না:**
- ইন্ডিগো বা নীল কালার প্রাইমারি হিসেবে ব্যবহার করবেন না
- ভারী গ্রেডিয়েন্ট বা নিউমরফিজম ব্যবহার করবেন না
- ইংরেজি টেক্সট ব্যবহার করবেন না (প্রযুক্তিগত শব্দ ছাড়া)
- তীক্ষ্ণ কোণ (sharp corners) ব্যবহার করবেন না
- অতিরিক্ত অ্যানিমেশন বা প্যারালাক্স এফেক্ট এড়িয়ে চলুন
